Frameworks, Libraries & Infrastructure Guide

Project: wardrobe-ai/ monorepo Version: 2.0 Date: June 22, 2026

1. Executive Summary

Multi-service, Docker-first application for AI wardrobe management, face authentication, outfit styling, 3D virtual try-on, e-commerce catalog, and social sharing.

LayerTechnologyRole
FrontendNext.js 15 + React 19UI, 3D viewer, camera auth
API GatewayNestJS 10 + TypeScriptREST API, auth, orchestration
AI ServicesFastAPI + PythonFace embeddings, clothing analysis
DatabasesPostgreSQL 16 + QdrantRelational data + vector search
InfrastructureDocker ComposeContainer orchestration

2. Architecture

Browser → Frontend (Next.js :3003) → Backend (NestJS :3001)
              ↓              ↓              ↓
         PostgreSQL      Qdrant      Face AI :8000 / Stylist :8001

Six Docker services on internal network wardrobe-network: frontend, backend, postgres, qdrant, face-service, stylist-service.
